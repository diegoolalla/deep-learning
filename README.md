# deep-learning
Tarea deep learning


La tarea de Deep Learning corresponde al módulo específico de la asignatura y debe realizarse utilizando Jupyter Notebook o Google Colab, trabajando directamente sobre las actividades incluidas en el archivo Examen.ipynb; el notebook puede ejecutarse en local o subirse a Google Colab y posteriormente descargarse, pero es imprescindible que todas las celdas estén ejecutadas, tanto las de código como las de texto. Las preguntas tipo test o de desarrollo deben responderse en la celda correspondiente, la cual también debe ejecutarse, y la puntuación de cada pregunta está indicada dentro del propio notebook. La entrega consiste únicamente en el notebook completamente ejecutado, cuyo nombre debe modificarse para incluir el nombre y apellidos del alumno separados por guiones bajos, siguiendo el formato indicado, por ejemplo Alberto_Ezpondaburu_Merino.ipynb.
